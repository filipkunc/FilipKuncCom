I had worked on a few projects in the past, especially hobby game development with my brother and I wanted to revive some of them and keep them alive.

## Space Warrior

A 2D side-scrolling platformer my brother and I built around 2010–2011 in
Visual Studio (Win32 + OpenGL 1.1 + 24-bit BMP sprites). The original code is
preserved almost verbatim — only the platform layer (window, input, sprites,
text) is replaced — and compiled to WebAssembly with Emscripten.

Play it here: [Space Warrior](/space-warrior).

Controls: arrows to move and jump, **Space** to use doors/switches/exits,
**T** to fire, **R** to cycle weapons, **P** to pause, **Esc** for menu.
Cheats from the original 2011 build still work — **N** skips to the next
level, **M** refills lives, **A** grants armor, **J** grants the jetpack,
**F** opens the key gate.
